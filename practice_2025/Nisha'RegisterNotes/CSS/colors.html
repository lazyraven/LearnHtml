    HSL & HSLA colors

    HSL stands for hue, saturation and lightenss. HSLA values are an extension of HSL with an alpha channel (opacity)

    Hue :  is a degree on the color wheel from 0 to 360.
            0 is Red
            120 is green
            240 is blue
        
    Saturation :  is a percentage value 0% means a shade of gray, and 100% is the full color.
    Lighteness : is also a percentage value, 0% is black and 100% is white.
                Ex: 
            hsl(0, 100%, 50%) => Red
            hsl(0, 100%, 25%) => Maroon (dark reddish-brown)
            hsl(240, 100%, 0%) => Blue
            hsl(0, 100%, 0%) => Black

    Shaded of Gray with HSL

                hsl(0, 0%, 20%)
            hsl(0, 0%, 40%)
            hsl(0, 0%, 60%)
            hsl(0, 0%, 90%)

    HSLA alpha

                hsla(0, 100%, 50%, 0.5)
            hsla(9, 100%, 64%, 0.8)

    HTML Style CSS

    Format the layout of a webpage

            Inline: style attribute inside HTML elements.
    
            Internal: style element in the head section
    
            External : by using a link element to link an external css file.
    
     
        Example 1 Inline: For display html code on ui write html code
        <h1 style="color: blue;">Title </h1>
        
        

        Example 2 Internal:
        <head>
         <style>
                body{
                    background-color: powderblue;
                }
                p{
                    color: red;
                }
         </style>
        </head>
        <body>
            <b>This is a paragraph>/b>
        </body>

        Example 3 External:
        <head>
        <link rel="stylesheet" href="color.css">
       </head>
       <body>
           <p>This is a paragraph>/p>
       </body>
