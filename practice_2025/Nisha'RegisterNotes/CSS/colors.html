    RGB(Red, Green, Blue)

    The intensity of the color with a value b/w 0 and 255, This means = 256*256*256 = 16777216 possible colors 

                red => (255, 0, 0)
            green => (0, 255, 0)
            blue => (0, 0, 255)

            black => (0, 0, 0)
            white => (256, 256, 256)

    X-Gray => eqal for all three parameter

                rgb(60, 60, 60)
            rgb(140, 140, 140)
            rgb(200, 200, 200)

    RGBA => (Red, Green, Blue, Alpha)

    Alpha => is a number b/w 0.0(full Transparency) and 1.0(not transparent at all)

                rgb(60, 60, 60)
            rgb(140, 140, 140)
            rgb(200, 200, 200)
